# Mode

Language mode: `full-en`

- Code (classes, functions, variables, inline comments, docstrings, debug messages): English.
- Docs & prose (README, CHANGELOG, reference files, commit messages, documentation prose): English.
- User-facing strings (UI labels, CLI messages, error messages): English.

## Exception

- `README.md` (English) has a French counterpart `README.fr.md`, kept in sync and fully translated, as a permanent derogation to the `full-en` mode.
